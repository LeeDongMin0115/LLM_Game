# 험프티 덤프티는 누가 죽였는가?

이 프로젝트는 **이상한 나라의 앨리스** 세계관을 바탕으로 만든 **대화형 추리 게임**입니다.  
플레이어는 이세계에서 온 탐정이 되어, 험프티 덤프티 사건의 진실을 조사합니다.

## 프로젝트 소개
- 피해자: 험프티 덤프티
- 용의자: 앨리스, 체셔 고양이, 흰토끼, 빌, 모자장수
- 플레이어 역할: 탐정
- 목표: 용의자들과 대화하며 단서를 모으고 진범을 찾기

## 주요 기능
- Streamlit 기반 웹 인터페이스
- OpenAI API를 활용한 캐릭터 대화 생성
- LangGraph를 활용한 조사 단계 및 게임 상태 관리
- 사건 노트 자동 정리
- 최종 추리 및 엔딩 분기

## 사용 기술
- Python
- Streamlit
- OpenAI API
- LangGraph
- python-dotenv
